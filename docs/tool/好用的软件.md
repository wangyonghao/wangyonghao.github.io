### 破解软件FAQ

打开任何来源

```bash
sudo spctl --master-disable
```
如已经开启任何来源，但依旧打不开
```bash
sudo xattr -d com.apple.quarantine /Applications/xxxx.app
```
按住 【control】 键，后在弹出的菜单上面，点击【打开】
