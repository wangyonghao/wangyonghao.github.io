### 由 oracle jdk 切换为open jdk 引发的图形验证码报错

相对于 Oracle JDK来讲 Open JDK 缺少了部分组件支持，并且重写了一些实现。在openJDK中，是不包含任何字体的，在生成图形验证码等使用awt.font的需求时，需要在服务器上安装fontconfig插件来引入字体类库。

安装字体插件 fontconfig

```
# CentOS
yum install fontconfig
# Debian / Ubuntu
apt install fontconfig
```
